# Mock Landing Page Showcase

A portfolio-quality fictional AI automation agency landing page built to showcase premium UI execution, responsive structure, motion, and clean component architecture.

Every brand, KPI, testimonial, price point, logo, and contact detail on the page is intentionally mock data created for demo and portfolio purposes only.

## Stack

- Next.js 16
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Project Purpose

This project is designed as a GitHub and Fiverr showcase piece:

- polished enough to feel like a real premium business website
- structured like a production landing page with reusable sections
- explicit about using fictional content so it does not misrepresent real brands or results

## Local Development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Production Checks

Run the quality checks used before deployment:

```bash
npm run lint
npm run build
```

## Deploying

This project is Vercel-ready.

Preview deployment with the Vercel CLI:

```bash
npx vercel
```

Production deployment:

```bash
npx vercel --prod
```

## Notes

- The site uses a dark premium visual system with reusable section components.
- All content is fictional and created only to demonstrate design, UX, and frontend implementation quality.
- Assets are code-driven with gradients, cards, iconography, and text-based mock brands instead of paid media.
